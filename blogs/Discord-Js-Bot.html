Getting Started

To get started we need to make a file i made mines named Discord Bot Blog but you can name it to anything.
Now we need to install discord.js by runing 

npm i discord.js



Now we need to set up our index.js by writing 
const Discord = require('discord.js');
const client = new Discord.Client({ 
intents: ['GUILDS', 
'GUILD_MEMBERS', 
'GUILD_MESSAGES',
 'GUILD_MESSAGE_REACTIONS'] });
let token = 'Your Token';
            
            client.on('ready', () => {
                console.log(`Logged in!`)
            });
            
            client.login(token)



Now we need to get our token by creating a new account! Lets go to Discord Dev Portals then make a new bot I will call mine Tutorial



So now we need make our bot by going to bot > create a bot



Now we need to add our bot and a command by doing 
const Discord = require('discord.js');
                    const client = new Discord.Client({ 
                        intents: ['GUILDS', 
                        'GUILD_MEMBERS', 
                        'GUILD_MESSAGES', 
                        'GUILD_MESSAGE_REACTIONS'] });
                        let token = 'token';
                            
                            client.on('ready', () => {
                                console.log(`Logged in!`)
                            });
                            
                            client.on('message', msg => {
                                if (msg.author.bot) return;
                                if (msg.content === 'ping') {
                                    msg.reply('pong');
                                }
                            })
                            
                            client.login(token)



now run 
node .
